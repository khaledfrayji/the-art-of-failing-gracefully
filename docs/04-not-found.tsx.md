# 🚫 not-found.tsx — Expected 404

## 📌 What It Is

Handles cases where data does not exist.

---

## 🧠 Trigger

```ts
notFound()
```

---

## ⚡ Behavior

* Renders UI for missing data
* Controlled, expected outcome

👉 It is NOT an error

---

## 📍 Scope

* Route segment only

---

## 🧠 Mental Model

> “Absence is not failure”

---

## 🎤 Teaching Line

> 404 is not a crash… it’s a valid state
